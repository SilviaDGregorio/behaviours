load

Load allos to load other DIA behaviours stored at the server.

The behaviours can be stored at the server just pressing save. At save time a name is selected. This is the name that has to be set when selecting this action.

Once the new behaviour is loaded, all current behaviour is lost, with exception of the LUA global variables.
